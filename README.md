This repository contains guidelines and other documents accompanying the specifications of the Asset Administration Shell and Submodel Template Specifications.

## Industrial Digital Twin Association (IDTA)

Governance of the specification is done by [IDTA](https://industrialdigitaltwin.org/en/).

## Documentation

There is one folder per document.

[asciidoc](https://docs.asciidoctor.org/), [PlantUML](https://plantuml.com/en/) and [antora](https://github.com/admin-shell-io/aas-specs-antora-ui) as well as [antora configuration](https://github.com/admin-shell-io/aas-guides-antora) are used to generate the documentation.
